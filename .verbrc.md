## Comparison

```js
// Multimatch
multimatch(['foo', 'bar', 'baz'], '!foo')).to.eql(['bar', 'baz']); // we didn't ask for these
// Globule
globule.match(['foo', 'bar', 'baz'], '!foo')).to.eql([]);
// Matched
matched(['foo', 'bar', 'baz'], '!foo')).to.eql([]);
```
